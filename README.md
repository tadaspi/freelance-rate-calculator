Calculate what you should **actually** charge as a freelancer. Most freelancers undercharge by 30-50% because they don't account for taxes, expenses, and non-billable time.

## The Problem

If you want **$80,000 take-home pay**, you can't just charge $40/hour for 2,000 hours.

You need to account for:
- **Self-employment tax**: 15.3% (Social Security + Medicare)
- **Income tax**: ~25% federal + state
- **Health insurance**: $6,000-$12,000/year
- **Business expenses**: Software, equipment, accounting
- **Non-billable hours**: 30-40% of your time goes to admin, marketing, invoicing

## The Formula

```
Required Revenue = (Desired Take-Home + Taxes + Expenses) / Billable Ratio
Hourly Rate = Required Revenue / Billable Hours Per Year
```
